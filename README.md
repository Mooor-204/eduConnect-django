# EduConnect - University Admission System

EduConnect is a comprehensive system that allows students to discover and apply to universities based on their education system and academic performance. The system will store information about various universities, their programs, requirements, and deadlines. The system will also allow students to search for universities by entering the education system they studied. If there are any universities available, they will be displayed. Students can then submit their applications.

## Features
- **3 Calculators** - Check your score for each education system
- **Apply to Universities** - One-click application via Google Forms
- **Track Applications** - See your application status (Submitted, Under Review, Approved, Rejected)
- **Prepare for Tests** - Access courses to prepare for placement tests
- **Profile Management** - Update your settings and privacy

## How to Run
1. Install Python and Django
2. Clone this repository
3. Run: `python manage.py migrate`
4. Run: `python manage.py runserver`
5. Open `http://localhost:8000`
